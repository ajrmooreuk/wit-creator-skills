# Creator Authority Skills — Video Creator Toolkit

**Go from zero audience to trusted voice in your niche using YouTube + Substack.**

8 skills you run in order. Each one builds on the last. No jargon, no fluff — just a repeatable system that works.

Originally built for the **Women in Tech Power Group**, but works for any creator in any niche.

---

## What You Get

| # | Skill | What It Produces |
|---|-------|-----------------|
| 1 | **Know Your Ground** | A one-page channel brief — who you are, who you serve, what makes you different |
| 2 | **Find Problems Worth Solving** | 20 ranked video ideas built from real audience pain points |
| 3 | **Titles & Thumbnails** | 3 scroll-stopping title variants + thumbnail concepts per video |
| 4 | **Script It — Hook, Help, Ask** | Complete video scripts with timestamps and B-roll markers |
| 5 | **Shorts Discovery Engine** | A 4-week Shorts calendar mapped to your long-form content |
| 6 | **Get Found — Simple SEO** | Optimised metadata + a reusable pre-upload checklist |
| 7 | **Build Your Community** | A daily 15-min + weekly 30-min engagement routine |
| 8 | **Make Money Before 10K** | A personal revenue plan with break-even target |

## Install

### Option A: Clone into your Claude Code skills directory

```bash
git clone https://github.com/ajrmooreuk/wit-creator-skills.git ~/.claude/skills/wit-creator-skills
```

### Option B: Clone anywhere and reference in your project

```bash
git clone https://github.com/ajrmooreuk/wit-creator-skills.git
```

Then add to your project's `CLAUDE.md`:

```markdown
## Skills
- Creator Authority Toolkit: `/path/to/wit-creator-skills/skills/`
```

---

## Use

Run the skills in order inside Claude Code:

```
/wit-creator-skills:pfc-mkt-channel-blueprint
```

Or if installed globally:

```
/pfc-mkt-channel-blueprint
```

Each skill tells you which one to run next when it finishes.

### The Pipeline

```
1. Channel Blueprint → 2. Content Ideation → 3. Titles → 4. Scripts → 5. Shorts
                                                                           ↓
                              8. Monetise ← 7. Community ← 6. SEO ←───────┘
                                                 ↓
                                       Feedback → back to Skill 2
```

## Your First 90 Days

| Week | What To Do | What You'll Have |
|------|-----------|-----------------|
| 1–2 | Run Skills 1–2 | Channel brief + 20 video ideas |
| 3–4 | Run Skills 3–4 | Title/thumbnail pack + first script |
| 5–8 | Film + publish 4 videos, set up Substack. Run Skills 5–6 | 4 videos live + Shorts calendar + SEO checklist |
| 9–12 | Run Skills 7–8 | Engagement routine active + revenue plan |

**Minimum viable schedule:**
- 1 long-form video per week
- 2–3 Shorts per week (repurposed from long-form)
- 1 Substack issue per week
- 15 min/day community engagement
