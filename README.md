## 3 Ways to Use This

### A. Google Docs (no tech required)
1. Open [WIT-Creator-Authority-Skills-Complete-v1.0.0.md](WIT-Creator-Authority-Skills-Complete-v1.0.0.md)
2. Copy contents into a Google Doc
3. Share with your group
4. Fill in the worksheets together or individually

### B. Claude Code — Step by Step
```bash
git clone https://github.com/ajrmooreuk/wit-creator-skills.git
```
Then tell Claude:
```
Read the skill at wit-creator-skills/skills/pfc-mkt-channel-blueprint/SKILL.md and run it.
My niche is Women in Tech.
```
Each skill tells you which to run next.

### C. Claude Code — Full Agent (all 8 at once)
```
Read and run the agent at wit-creator-skills/agent-creator-authority.md

My niche: Women in Tech. I'm a senior cloud engineer who mentors
junior women through career transitions.
```
Produces all 8 artefacts in one session.

## Your First 90 Days

| Week | Focus | You'll Have |
|------|-------|------------|
| **1–2** | Skills 1–2 | Channel brief + 20 video ideas |
| **3–4** | Skills 3–4 | Title/thumbnail pack + first script |
| **5–8** | Film 4 videos + Substack. Skills 5–6 | 4 videos live + Shorts calendar + SEO checklist |
| **9–12** | Skills 7–8 | Engagement routine + revenue plan |

**Weekly schedule:**
- 1 long-form video (Tue or Thu)
- 2–3 Shorts (repurposed from long-form)
- 1 Substack issue
- 15 min/day community engagement
